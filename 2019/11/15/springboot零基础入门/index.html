springboot零基础入门

Spring Boot是由Pivotal团队提供的全新框架，Spring Framework也是他们的杰作，Spring最早是由Rod Johnson和Pivotal团队开发的。在开发Spring Framework之前还著书《Expert One-on-One J2EE Development without EJB》用于批判当时业界主流后端框架EJB，缘由是EJB过于臃肿，这本书为Spring Framework的诞生奠定了理论基础。由于Rod Johnson在Spring Framework的理论和实现上的卓出贡献，其被称为Spring之父。—为大佬献上狗头

Spring Boot + Mybatis实现xml版本的增删查改demo

​ 以下代码可以参考我的github https://github.com/lonelyProgramMonkey/SpringBoot-demos/tree/master/springboot_mybatis_xml ,如果觉得本文对你有帮助的话，还请star一下我的github，你的支持就是对我最大的鼓励。

​ 我们首先从增删查改的demo来入门，hello world就不赘述了。首先你需要配置一个数据库进行数据的增删查改。这里我推荐如今的主流数据库MySQL，具体可以参考菜鸟教程https://www.runoob.com/mysql/mysql-tutorial.html 进行安装。

​ 安装MySQL后导入users.sql https://github.com/lonelyProgramMonkey/SpringBoot-demos/blob/master/springboot_mybatis_xml/users.sql 到MySQL的test数据库来建表，具体可以使用MySQL命令行、Navicat、MySQL workbench来导入，推荐使用前两种方式。
